Media Server API Documentation

This API provides endpoints to manage a collection of movies, TV series, and songs.

Available Endpoints

GET /:type

Get all items of a specific type (movies, series, or songs)

Example: GET /movies

POST /:type

Add a new item to a specific type

Example: POST /movies

Request body: 
{
    "title": "Inception",
    "director": "Christopher Nolan",
    "year": 2010
}

PUT /:type/:id

Update an existing item by ID

Example: PUT /movies/1

Request body: 
{
    "year": 2011
}

DELETE /:type/:id

Delete an item by ID

Example: DELETE /movies/1

Data Types

Movie

{
    "id": number,
    "title": string,
    "director": string,
    "year": number
}

Series

{
    "id": number,
    "title": string,
    "seasons": number,
    "creator": string
}

Song

{
    "id": number,
    "title": string,
    "artist": string,
    "year": number
}
